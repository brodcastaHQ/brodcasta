# Nexios Framework Features

## Key Features

- **Async/Await Native** - Built from ground up for Python async/await patterns
- **TypeScript Support** - Full type safety with modern JavaScript
- **FastAPI Integration** - Leverages FastAPI's high performance
- **WebSocket Support** - Real-time bidirectional communication
- **Server-Sent Events** - Efficient one-way data streaming
- **JWT Authentication** - Secure token-based authentication
- **CORS Handling** - Built-in cross-origin resource sharing
- **Tortoise ORM** - Async ORM with multiple database support
- **Docker Support** - Container-ready configuration
- **Hot Reload** - Automatic code reloading during development
